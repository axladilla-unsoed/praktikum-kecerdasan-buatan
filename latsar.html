Deskripsi Mata Kuliah

Praktikum dirancang untuk memperkuat pemahaman konseptual melalui implementasi langsung pada notebook interaktif.

Mata kuliah ini bertujuan memberikan pemahaman praktis mengenai konsep dan implementasi Kecerdasan Buatan (Artificial Intelligence) menggunakan Python. Mahasiswa akan mempelajari penerapan Logika Fuzzy, Sistem Pakar, dan Algoritma Genetika melalui praktik di notebook interaktif.

Artificial Intelligence Python Programming Fuzzy Logic Expert System Genetic Algorithm

Interaktif

Mahasiswa dapat menjalankan, menguji, dan memodifikasi kode secara langsung pada notebook.

Terstruktur

Materi dibagi menjadi beberapa topik inti sehingga alur belajar lebih sistematis dan mudah diikuti.

Relevan

Setiap topik diarahkan pada penerapan konsep AI untuk problem solving secara praktis.

Tujuan Praktikum

Setelah mengikuti praktikum ini, mahasiswa diharapkan tidak hanya memahami teori, tetapi juga mampu mengimplementasikannya.

Menguasai dasar pemrograman Python sebagai fondasi untuk membangun solusi komputasional pada praktikum Kecerdasan Buatan.
Memahami dan mengimplementasikan Logika Fuzzy untuk memodelkan keputusan yang bersifat tidak pasti.
Mampu membuat sistem pakar sederhana berbasis aturan dengan memanfaatkan knowledge base dan inference engine.
Mengimplementasikan Algoritma Genetika untuk mendukung proses pencarian solusi pada berbagai permasalahan optimasi.
